## üé® Technical Architecture

### CSS Animations Added

1. **floatChocolate** (8-18s)
   - Translates and rotates chocolates
   - 4-point keyframe animation
   - Opacity varies 0.8-1.0

2. **deliveryMove** (3s)
   - Truck moves left to right
   - Smooth ease-in-out
   - Infinite loop

3. **floatTeddy** (10-22s)
   - Complex 3-point animation
   - Rotation + scale + translation
   - Opacity 0.7-0.9

4. **floatRandom** (10-25s)
   - 4-point flower/heart movement
   - Rotation 0-360¬∞
   - Opacity 0.6-1.0

5. **floatQuote** (15-35s)
   - Horizontal scroll across screen
   - Fade in/out at edges
   - Linear timing

6. **floatHeart** (8-20s)
   - Vertical + scale + rotation
   - 2-point keyframe
   - Opacity 0.6-1.0

7. **fallSparkle** (2-5s)
   - Top to bottom fall
   - 360¬∞ rotation
   - Fade in/out

8. **explodeSparkle** (2s)
   - Radial explosion from center
   - Scale 0 ‚Üí 1.5 ‚Üí 0.5
   - Opacity 1 ‚Üí 0

9. **heartPulse** (1s)
   - Scale 1.0 ‚Üí 1.2 ‚Üí 1.0
   - Infinite loop
   - Ease-in-out

### Responsive Design

**Mobile Optimizations:**
- Font sizes: Responsive with `text-xl md:text-2xl` patterns
- Padding: Adjusted for smaller screens
- Floating elements: Reduced count on mobile (media query ready)
- Touch-friendly: All buttons have adequate tap targets
- No horizontal scroll: `overflow-x: hidden` on containers

**Music Player Clearance:**
- All pages: `padding-bottom: 150px`
- Main content: `min-height: 70vh`
- Z-index hierarchy: Background (1), Content (2), Music Player (100)
- Fixed positioning: Music player at bottom with z-index 100

### Performance Considerations

**Optimizations:**
- Debounced animations: Stagger delays prevent simultaneous starts
- Cleanup: `setTimeout(() => element.remove(), duration)` on all dynamic elements
- CSS animations: Hardware-accelerated (transform, opacity only)
- Reduced motion: Can add `prefers-reduced-motion` media query if needed
