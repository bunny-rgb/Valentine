# Day 7 Music

This folder should contain the Day 7 (Valentine's Day) music files.

**Note:** Due to size constraints, Day 7 music files are currently streamed from:
- **GitHub CDN**: https://raw.githubusercontent.com/bunny-rgb/Valentine/main/public/static/music-day7/

The files were uploaded to GitHub repository but may not be in the current working directory.

## Files
- 54 MP3 files (~365 MB total)
- Uploaded in 9 batches to GitHub

## Usage
The app.js file is configured to stream these files directly from GitHub, so local copies are not required for deployment.
